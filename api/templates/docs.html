User

API REST de cadastro de autenticação de usuários usando tokens **JWT** como meio de autenticação.

POST Register
localhost:8000/api/register

Recebe as informações do usuário em formato JSON com os seguintes campos: name, cpf e password. O campo role é obrigatório. Se o campo role for "C", o usuário será comum. Caso a role for igual a “A”, o usuário será administrador

BODY raw
                                                  
                                                    
{
    "name": "Levy",
    "cpf": "09627849810",
    "password": "lima2409",
    "role": "C"
}
Example Request
  • Register
                                                
POST localhost:8000/api/register

{
    "name": "Levy",
    "cpf": "09627849810",
    "password": "lima2409",
    "role": "C"
}
POST Login
localhost:8000/api/login

O usuário autenticado receberá um token JWT de acesso e poderá acessar o endpoint de usuários.

BODY raw
                                                  
                                                    
{
    "cpf": "05012119266",
    "password": "lima2409"
}
Example Request
  • Login
                                                
POST localhost:8000/api/login

{
    "cpf": "05012119266",
    "password": "lima2409"
}
GET User Page
localhost:8000/api/users

Todos os usuários autenticados poderão visualizar os usuários cadastrados.

Example Request
  • User Page
                                                
GET localhost:8000/api/users
PATCH Change user
localhost:8000/api/users/edit/2

Somente usuários admin poderão acessar esse endpoint. O administrador deverá indicar o id do usuário a ser editado via query string e enviar as mudanças no corpo da requisição.

BODY raw
                                                  
                                                    
{
    "name": "Marcinha",
    "password": "marcinha123",
    "cpf": "05012119266"
}
Example Request
  • Change user
                                                
PATCH localhost:8000/api/users/edit/2

{
    "name": "Marcinha",
    "password": "marcinha123",
    "cpf": "05012119266"
}
